## Commands

install dependencies

```
pnpm i
```

run only this project

```
pnpm dev
```

run this project with host

```
pnpm build
```

```
npx serve dist -p 5004
```

run tests

```
pnpm test
```

```
pnpm test:e2e
```

First build the micro-fronts repos and run this project last.
